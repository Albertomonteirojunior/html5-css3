Aprendendo usar Pseudo class

As pseudo classes configura um estado. e seu paremtro é : , exemplo:


        div:hover {
            background-color: yellow;
        }

Está configurado com o estado houver, quando passar o mouse passar por cima a div ficará amarela.

Usando pseudo classes

Usando filho dentro das pseudo classes. EX:


    div:hover > p {
    display:block;
    color: white;
    background-color: red;
    width: 200px;
}
