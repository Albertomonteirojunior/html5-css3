Usando fontes externas

Usando google Fontes

Site: Site de fontes do Google.

Para importar as fontes do Google fontes é preciso acessar o site escolher uma fonte clicar em add style. depois selecionar @import e copiar tudo que está entre stlye /style


    @import url('https://fonts.googleapis.com/
    css2?family=Fredericka+the+Great&family=League+Spartan&family=
    Nanum+Pen+Script&family=Prata&family=Sanchez:ital@1&family=
    Shizuru&family=Smooch+Sans&display=swap');

Depois selecionar o codigo css do fon-family

Ex:

font-family: 'Shizuru', cursive;

Alinhando texto em Css

3 tipos de alinhamentos

Temos os alinhamentos a esquerda, direita e ao centro. Para isso usamos os parametros na Css


    h1 {
        text-aling:center:
    }

    h2{
        text-aling:rigth;
    }
     p {
         text-aling: justify;
     }

Para criar o alinhamento do paragrafo usamos o Parametro:


    p {
        text-indent: 30px;
    }
