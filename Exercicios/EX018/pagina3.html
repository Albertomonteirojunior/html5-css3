Usando Font-face

Para usar fontes internas, precisa ir até um site, fazer download dos arquivo da fonte, depois descompactar o arquivo e colocar ele dentro da pasta do arquivo index do seu site.

Parametro para inserir a fonte ao site



@font-face {
    font-family:'Terror' ;
    src: url('Fonts/Frightmare.ttf') 
    format('truetype');
           }

Na URL inserir o caminho onde a fonte foi instalada, no paramentro font-family dar um nome para fonte. esse nome sera usado para inserir a fonte onde desejar

Tipos de Format

  • opentype (otf)
  • truetype(ttf)
  • embedded-opentyoe
  • truetype-att(apple advanced typography)
  • svg
