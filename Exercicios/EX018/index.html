Tipo de Fontes

O tamanho de todas as letras minusculas são baseadas no X minusculo (altura x).
Temos a altura das fontes maiusculas. e temos as fonte minusculas que ultrapassam o tamanho da fonte X , são chamadas de acendentes, temos as letras que vazou para baixo chamado de descendente

E todo esse conjuto é chamado de corpo

  • normal - ( n )
  • bold - ( pesado )
