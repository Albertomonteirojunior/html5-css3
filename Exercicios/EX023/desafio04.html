Configuração de colunas izoladas

Para selecionar uma coluna existe uma nova tag chamada colgroup.

EX:


            colgroup
                col class="cnome"
                col class="csexo"
                col class="cidade"
                col class="cprof"
           /colgroup

Para usar ela, precisa estar dentro de table depois usar a tag colgroup em seguida criar as tags cole dar uma class ou id para elas.
