Criando Tabelas

Todas as tabelas simples possuim uma hieraquia

  • Table = tabela
  • Table ROW (tr) = linha de tabela
  • Table READER = cabeçalho de tabela
  • Table DATA (td) = dado de tabela

Para deixar as bordas juntas da tabela usar esse parametro:


    
td { 
  border-collapse: collapse;
    }
A1 B1 C1
A2 B2 C3
A3 B3 C3
A4 B4 C4

Temos uma propiedade chamada de vertical aling . temos o seguintes parametros dentro dessa propiedade

  • Vertical-aling = alinhamento vertical
  • Top = alinha acima
  • middle = alinha ao meio
  • bottom = alimaneto abaixo

    style

    td {
    vertical-align: bottom;
       }

   /style

Tags semanticas para tabelas

Uso da tabelas grandes exemplo de estrutura semantica:

  • CAPTION = legenda
  • THEAD = cabeça da tabela
  • TBODY = corpo da tabela
  • TFOOT = Rodapé da tabela
