Configuração

Para alinhar uma imagem dentro de uma box, utilisiza-se os seguintes paremetros:




            #container
                {
                background-size: 100% 100%;
                background-image: url('imagens/target001.png');
                position: relative;
                height: 96vh;
                padding: 10px;
                background-color: purple;
            
                }
            #conteudo
                {
                background-size: 100% 100%;
                background-image: url('imagens/target001.png');
                position: absolute;
                width: 400px;
                height: 200px;
                background-color: yellow;
                left: 50%;
                top:50%;
            
                transform: translate(-50%,-50%);
                }

Para fazer essa configuração precisa usar esse dois parametros: Dentro do conteudo precisa ser position: absolute; e depois usar o transform: translate(-50%,-50%) para fazer os ajustes da posição

Position

Colocando o conteudo como posição absoluta dentro de um container ou caixa (box) e possivel movimentar e alinhar ele dentro da caixa( container).

Transform:translate ( )

É possivel movimentar a caixa via porcentagem para fazer o alinhamento. como mostra a imagem no inicio.
